**Student Registration System**

A simple and responsive Student Registration System built using HTML, CSS, and JavaScript.
Students can register their details, and the system allows editing, deleting, and storing records using localStorage, so data stays even after refreshing.

**Features**

- Add new student records
- Edit existing records
- Delete records instantly
- Live validation (name, ID, email, contact)
- Prevents duplicate Student IDs
- Auto-scrolling table with dynamic vertical scrollbar
- Fully responsive (mobile, tablet, desktop)
- Data saved in browser localStorage
- Clean UI with glassmorphism +  gradient theme

**Tech Used**

- HTML5
- CSS3 (Glassmorphism UI + responsive layout)
- JavaScript (ES6)
- LocalStorage for persistence

**How It Works**

- Fill in student details in the form
- Click Add Student
- Records appear in the table below
- Use Edit or Delete buttons for each record
- All data is auto-saved and restored after page refresh
